# VG-Canvas

A drawing app with real-time collaboration.

## Quick Start

```bash
# Install dependencies (once)
npm install

# Start app
./vg-canvas.js
```

## Collaboration

1. Choose option 2
2. Click "Collaborate" button
3. Enter name
4. Create/join room
5. Share room ID
